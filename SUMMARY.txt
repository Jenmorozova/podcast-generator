🎯 СЛЕДУЮЩИЕ ШАГИ:

1. ДЕПЛОЙ (5 минут):
   
   $ vercel login
   $ npm run deploy
   
   Скопируйте Production URL

2. СОЗДАЙТЕ БОТА (3 минуты):
   
   - Откройте @BotFather в Telegram
   - /newbot → введите название
   - Сохраните токен

3. СОЗДАЙТЕ MINI APP (2 минуты):
   
   - /newapp в @BotFather
   - Выберите бота
   - Вставьте Production URL
   
4. ГОТОВО! 🎉
   
   Откройте бота → нажмите меню → используйте!
